# Delicious Bistro — Restaurant Website

A fully responsive restaurant website built with HTML, CSS, Bootstrap, and JavaScript.

## Features
- Responsive design across all devices
- Interactive menu with categories
- Table booking form with email notifications
- Contact form with instant delivery
- Smooth scroll animations (AOS)
- Gallery section

## Tech Stack
- HTML5, CSS3, JavaScript
- Bootstrap 5
- AOS (Animate On Scroll)
- Web3Forms (no backend needed)

## Forms Setup
Both the **Book a Table** and **Contact** forms are powered by [Web3Forms](https://web3forms.com).
